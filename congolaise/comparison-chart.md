# The Three Standards — Comparison Chart

*Metropolitan Nelôxi · Saharannaise · Congolaise, laid side by side.* One language, three seats:
the grammar is identical in every column, and the vocabulary carries three worlds. This chart is the
whole pluricentric shape on one page — read a row across to see exactly where the standards agree
(the shared spine, the Level-1 crust) and where they diverge (the Level-3 domestic core, three
different ways). Metropolitan forms are coursebook canon; Saharannaise forms are archived under
`../saharannaise/`; Congolaise forms are archived in lexicons R/P/G. A dash (—) marks a genuine
**gap** — a thing one world has no native word for, which is itself part of the story.

---

## 0 · The shared spine — identical in all three

This is what makes them one language, not three. None of it varies by seat.

| Feature | All three standards |
|:---|:---|
| word order | SVO |
| negation | pre-verbal **äb** |
| copula | **um / ūli** |
| imperative / hortative | bare stem / **-m** |
| past tense | **-i** (two classes, §120) |
| object · adessive · ablative · goal · comitative | **-n/-ôn** · **-al** · **-lt** · **-lô** · **-k** |
| agent · derivation | **-ôr**, **-in**, **-mus**, **-stā**, **-ldā**, causative **-tā**, comparative **-mi** |
| number · clock · calendar | dozenal · 24-hour · Low German |

> **Level 1, shown:** *Provinsô agā tri rīgô* — "the province has three regions" — is the **same
> sentence** in all three standards. The spine and the Romance learned crust are common ground.

## 5 · The signature words — each world's own

The clearest proof that these are three living lexicons: each seat has words the others simply lack,
because each names a world the others do not live in.

| Only in… | Word | Gloss |
|:---|:---|:---|
| **Saharannaise** | **žəml** | camel |
| **Saharannaise** | **gāfila** | caravan |
| **Saharannaise** | **ṭāyf** | sandstorm |
| **Congolaise** | **jeli** | griot (hereditary oral-historian) |
| **Congolaise** | **terānga** | hospitality (the guest-honor ethic) |
| **Congolaise** | **wālo / jēri** | flood-recession land / rainfed dryland |
| **Congolaise** | **kora** | the 21-string harp-lute |

These are the **back-flow candidates** in both directions: the Saharan *gāfila* has already ridden
up the trade routes into the mainland lexicon; the riverine *jula, kora, terānga* are flagged to
follow. A word enters another seat's canon only when that seat merges it — never automatically.

## How to read the divergence

- **Row-by-row agreement is Level 1 / shared-spine** (row 0; *sūr* "big"; *bīr* "well"). The
  standards read each other with ease here.
- **Row-by-row divergence is Level 3 / domestic core.** *nīt / līl / jemma* "night"; *veç / aman /
  ndiyam* "water." A mainlander is lost; each coastal speaker, schooled in the standard, still reads
  the mainland line — the one-way asymmetry.
- **A dash is a gap, and gaps carry the history.** The mainland has no camel; the river has no
  camel either but has a griot the others lack. What each language *cannot* say maps its world as
  sharply as what it can.
- **The Congolaise column names its tributary** (last column) — because Congolaise's own rule is
  *which* Niger-Congo well a root comes from (`CONFLUENCE-PRINCIPLE.md` §1). Pulaar owns the
  domestic bulk; Soninke-Mande owns gold, salt, trade, road; Wolof owns fish, market, the coast.

*The doctrine behind the Congolaise column is `CONFLUENCE-PRINCIPLE.md`; the tiers are shown in
prose in `showcase-trilect.md` and `reader-flood.md`; the Saharan column is archived under
`../saharannaise/`; the Metropolitan column is the coursebook (`../coursebook/`).*
