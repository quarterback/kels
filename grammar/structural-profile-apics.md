# Nelôxi — Structural Profile (APiCS parameters)

*A typological profile of Nelôxi answered against the **APiCS** framework — the Atlas of Pidgin
and Creole Language Structures (Michaelis, Maurer, Haspelmath & Huber, MPI-EVA). Every value is
grounded in the foundation grammar modules and the coursebook; where canon has not decided a
feature it is marked **⟨open⟩**, not invented. Daughter-lect/register divergences (Saharannaise,
inland, metrolect, Tristine) are flagged inline and summarized at the end.*

*The headline: Nelôxi is a **designed multiple-lexifier contact language** — Livonian-Karelian
Baltic-Finnic grammar carrying a five-quarry loan lexicon. Structurally it patterns like its
Finnic substrate (rich agglutinative noun morphology, near-zero verb inflection, postpositions,
no articles, no gender, dozenal counting), which makes it an unusual entry against a database
dominated by Atlantic/Romance-and-English-lexified creoles.*

## 1 · Word order

| Parameter | Nelôxi | Source / note |
|---|---|---|
| Subject–Object–Verb order | **SVO**, fixed | coursebook Part One; all examples |
| Adposition & NP | **Postpositional** (noun in linking `-s`, word follows: *kōtôs al* "under the house") | module 08 |
| Adjective & noun | **Adj–N** (adjective precedes; invariant) | module 10 |
| Numeral & noun | **Num–N** (*tri kalāt* "three fish") | module 12 |
| Demonstrative & noun | **Dem–N** (*se vana livrô*) | module 07 |
| Possessor & possessum | **Possessor–Possessum**, genitive `-s` precedes (*kōtôs port*) | modules 06, 08 |
| Relative clause & noun | **Both**: prenominal participle (*kantāji mēs* "the singing man") or postnominal `ken/mis` clause (*mēs, ken kantāi*) | module 07; VERB-REF |
| Degree word & adjective | degree precedes; comparative is suffixal `-mi` (below) | module 10 |

Consistent Finnic-type profile: SVO clause with a head-final, dependent-marked noun phrase.

## 2 · Nominal categories

| Parameter | Nelôxi | Source / note |
|---|---|---|
| Definite article | **None** | module 06 (bare noun) |
| Indefinite article | **None** | — |
| Nominal plural marking | Suffixal **`-d`**, before case endings (*kōtôd*, *kōtôdôl*) | module 06 |
| Plural obligatory? | **No** — counted nouns take **partitive singular**, not plural (*tri kalāt*, never *tri kalād*) | module 12 |
| Associative plural | ⟨open⟩ — no dedicated construction attested | — |
| Grammatical gender / noun class | **None** | module 07 (*tä* = he/she/it) |
| Case | **Eight cases** + plural: nominative, object `-n`, partitive `-t/-tô`, linking/genitive `-s`, place `-l`, goal `-lô/-lä`, source `-lt`, comitative `-k` | module 06 |
| Adnominal possession | Juxtaposition with genitive `-s` on possessor (*mäs nim* "my name") | modules 07, 08 |

| Comparative | Suffix **`-mi`** + *kui* "than" (*raskmi kui…* "heavier than…") | module 10 |
| Superlative | **Frame**, not affix: *kôust* "of all" | dictionary (`kôust`) |

## 4 · Verbal categories (TMA, voice, mood)

| Parameter | Nelôxi | Source / note |
|---|---|---|
| Subject agreement | **None** — verbs are isolating/invariant | VERB-REF §1 |
| Tenses | **Two**: present (bare stem), past **`-i`** | module 09 |
| Future marking | **None** — bare present + a time-word (*demā mä lǟdô*) | module 09 |
| Perfect / pluperfect | **None** — plain past, + *jubā* "already" if needed | module 09 |
| Progressive / imperfective | **No verb form** — aspect lives on the **object's case** (partitive `-t` = ongoing) | modules 09, 01 |
| Habitual | Bare present + frequency word (*sagõ* "often") | module 09 |
| TMA expression locus | **Split**: past on the verb; futurity on time-words; boundedness/aspect on the object case | module 09 |
| Preverbal TMA/modal particles | **Yes** — analytic modals precede the plain verb: *vôī* "can," *pidā* "must" | VERB-REF §3 |
| Conditional/irrealis | Suffix **`-ks`** (*voleks* "would want") | modules 09, VERB-REF |
| Copula (nominal / adj / locative predication) | **Single copula `um`** (pres) / `ūli` (past), person-blind; used for all three (*nim um Ron*; *livrô um vana*; *famīl·la um kōtôl*) | module 09; VERB-REF |
| Passive | **Participial only** — `-dôt` (*donādôt livrô* "the given book"); no dedicated passive voice | VERB-REF §4 |
| Participles | Active/agent **`-ji`**; perfect/passive **`-dôt`** | VERB-REF §4 |
| Serial verb constructions | **None** — modality is analytic-preverbal, not serial | VERB-REF |
| Verb derivation | Rich: causative/frequentative/momentane/reflexive lanes; denominal `-stā`, `-ldā` | module 02 |

## 10 · Daughter-lect & register divergences (structural only)

The grammar (sections 1–7) is **invariant across all varieties** — this is the whole design. Only
the following diverge, and only in the places named:

- **Saharannaise** (Atlantic co-standard) — **identical structure**; diverges by *lexicon* (domestic
  core rerooted to Hassaniya Arabic + Zenaga Berber) and by an *added phonological scar set* (ə, ž,
  ġ, emphatics ṭ ṣ ḍ ẓ; the §47 friction ḥ/š/kh → x). Same cases, TMA, negation, numerals.
- **Inland** (*Nurôkēļ*) — **numerals**: retains the withdrawn Finnic decimal set (*üks, kaks, kolm*)
  where the standard is Slavic-dozenal; otherwise more conservative morphophonology (no clipping).
- **Metrolect (harbor end)** — phonological only: final unstressed `-ô` drops, geminate `l·l → l`.
- **Tristine** (Adriatic dialect) — lexical/register tilt (courtesy + port-legal-commercial Romance-
  Venetian); one sanctioned morphological bend, the participle *registrǟd* off standard `-dôt`.

## Open slots (undecided in canon — do not invent)

Associative plural · dedicated focus/cleft construction · "no longer/anymore" (flagged gap,
module 09) · postpositional "until/since/beside" (module 08 gaps) · `vôīks` "could" (predicted,
unattested, VERB-REF §3). A contributor who needs one flags it for the Rector.
