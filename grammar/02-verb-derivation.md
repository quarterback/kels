# Nelôxi Grammar — The Verb Derivation Engine
*Foundation module 02 · charter §60 · canon v5.7*

One verb root generates a family of verbs through four productive chains. **Derive, don't
coin** — if you need "to feed," "to keep glancing," "to come loose," build it from the chain.

## The four chains

### Causative — *-tā / -tä* — "make/cause X, have X done"
Adds an agent who causes the action.
- *kūlô* hear → *kūlôtā* announce (make hear)
- *sȫ* eat → *sȫtä* feed (make eat)
- *kadô* vanish → *kadôtā* get rid of (make vanish)

### Frequentative — *-ilā / -elä* — "keep doing X, do X repeatedly"
Marks repeated, habitual, or drawn-out action.
- *lôigā* cut → *lôigilā* keep cutting
- *nǟgô* see → *nǟgilä* keep looking, gaze
- *koput* knock → *koputelä* keep knocking

### Momentane — *-ahtā / -ähtä* — "do X once, suddenly, in a burst"
The opposite of the frequentative — a single instant of the action.
- *naurô* laugh → *naurahtā* give a laugh
- *nǟgô* see → *nǟgahtā* glance
- *līkô* move → *līkahtā* give a start, twitch

### Reflexive / automative — *-u / -y* — "X happens of itself; do X to oneself"
Removes the external agent; the action turns back on the subject or happens spontaneously.
- *avā* open (something) → *avau* come open
- *kôrdā* fix → *kôrdau* get fixed, come right
- *pesô* wash → *pesu* wash oneself

## The chains stack

- *sȫ* eat → *sȫtä* feed → *sȫtätä* have (someone) fed
- *avā* open → *avau* come open → *avautelä* keep coming open

## Rule for contributors

Need a verb that is a variant of an existing one — caused, repeated, momentary, or
self-happening? **Apply the chain to the existing root.** Only coin or borrow a genuinely new
root when no existing verb is the base.
