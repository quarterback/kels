# Contributing to Nelô kēļ

All contribution rules live in `college/kels-kolegi-charter.md`. Read it whole before coining a single word.

The short version:
- Work one domain batch at a time (25–40 words), in the charter's submission format, with collision notes.
- Check every candidate against `data/headwords.txt` (regenerated after each merge).
- Old-stratum phonotactics by default; late-stratum is a thin crust (see charter §11 and the ratio rulings).
- Examples may use only canon vocabulary.
- Nothing is canon until the Rector merges it into `coursebook/nelo-kel-coursebook.md`.
