# Nelôxi Agent Bundles

Pre-assembled, self-contained documents. Hand an agent ONE bundle matching its task — not the
repo, not the charter. Each bundle contains everything that agent needs and nothing extra.

- **BUNDLE-domain-agent.md** — for coining vocabulary in a domain (plants, tools, law, etc.).
  Contains: the delegate brief, the function-word cheat-sheet, verb-derivation, compounding, and
  the full current headword list.
- **BUNDLE-corpus-agent.md** — for writing a genre text (folktale, letter, dialogue, etc.).
  Contains: the corpus brief (with embedded grammar + word list), the cheat-sheet, tense/aspect,
  pronouns, and adjectives.
- **BUNDLE-grammar-reference.md** — the complete grammar in one file. Hand to a grammar agent, or
  to ANY agent that has started drifting (using Estonian, guessing forms) to re-ground it.

## Keeping them current

Bundles embed a snapshot of the headword list and grammar. Regenerate them after canon changes:
run the bundle-build step (see the Rector handoff) so a fresh agent never gets a stale list.
The authoritative source is always the coursebook; bundles are convenience assemblies.
