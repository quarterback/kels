# The East Fork — the Turkic Loan Quarry

*The everyday layer. Turkic plays here the exact structural role Low German plays in
Metropolitan: the heaviest source of casual, lived, market-and-kitchen borrowed vocabulary,
sitting as **doublets** beside the native Nelôxi core words — never replacing them. The
clerk writes* rā, *the bazar says* para*; both are canon in this seat, exactly as* melk
*trades beside* lēt *on the mainland. Turkic leads the loan layer and only the loan layer:
the grammar lead is Circassian (`grammar-circassian.md`), and the pan-Turkic bloc next door
owns the Turkic identity — a fork that borrowed Turkic structure would blur into it, while a
fork that haggles in Turkic loanwords just sounds like the east. Lighter sources where the
map puts them: **Persian/Khorasani** at the Herat end, **Genoese-Italian** trade residue at
the Crimean end.*

---

## §1 · Adaptation rules (closed set)

Turkic loans are written with the shared Metropolitan orthography — **no new graphemes**
(`script.md`). The mappings:

| Source | East-fork spelling | Example |
|---|---|---|
| ç [tʃ] | **tx** (compositional: t + x [ʃ]) | *çay → txai*, *çok → txok*, *çorba → txorbā* |
| ş [ʃ] | **x** (the §47 friction rule) | *şeker → xekēr*, *yavaş → yavax* |
| c [dʒ] | **dž** (the mainland scar, already sanctioned) | *cep → džep* "pocket" |
| ı [ɯ] | **ô** [ɤ] (nearest native vowel) | *kısmet → kôsmet* |
| ö ü | **ö ü** (native letters) | *dükkân → dükān* |
| ğ | lost; lengthens the vowel → macron | *dağ → dā* (unused — see §4) |
| long vowels (Tk./Pers.) | macron | *hesap → hesāp*, *nān* |
| word-final -e | **-ē** | *kahve → kahvē* |

The set is closed: a loan may not invent orthography outside this table without a Xasā
ruling.

## §2 · The everyday doublets (casual ↔ core)

Scoped to **casual register**. The left column is bazar speech; the right column is the
same meaning in coursebook canon, which remains the written/formal word. Tag: all left-column
forms are **(Id.)**.

| Casual (Id.) | Core (canon) | Meaning | Source |
|---|---|---|---|
| **para** | rā | money | Tk. *para* (itself Persian *pāre* — the word worked this route both ways) |
| **bazar** | turg | market | Tk. *pazar* / Pers. *bāzār*; *markôt* stays the formal trade-word — a three-register row |
| **txai** | tē | tea | Tk. *çay*; live back-flow candidate (Id.→Metr.) via Uusatôm |
| **ekmek** | pa | bread | Tk. *ekmek* |
| **nān** | pa | bread (Herat end) | Khorasani Pers. *nān* — regional beside *ekmek*; the two casuals split the map |
| **dost** | amīk | friend, mate | Tk./Pers. *dost* — spans the whole span, Crimea to Herat |
| **kôsmet** | merjagô | fate, the dealt share | Tk. *kısmet*; the bazar's fate-word beside the sea's (*fatômô* stays bookish) |
| **kahvē** | kafē | coffee | Tk. *kahve* — the doublet that meets itself: both ultimately the same Arabic word, arrived by opposite coasts |
| **xekēr** | sukrô | sugar | Tk. *şeker* — likewise a cousin-doublet (both from the same old root) |
| **dükān** | butīk | shop, stall | Tk. *dükkân* |
| **hesāp** | kontô | the bill, the account | Tk. *hesap*; *hesāp bidô!* "the bill, please!" |
| **džep** | — | pocket | Tk. *cep*; gap-filler, no core rival |
| **txok** | mult | very, much | Tk. *çok*; casual intensifier |
| **yavax** | lent | slow; easy, gently | Tk. *yavaş*; reduplicated *yavax-yavax* "easy now, no rush" — the *nūnū* pattern with a Turkic face |
| **haidē** | — | come on!, let's go | the Black Sea areal word (Tk. *hayde*, shared with the whole littoral); discourse particle, no core rival; live back-flow candidate (Id.→Metr.) |

## §3 · Realia — named, not translated

The compound ban, East edition: no Nelôxi calque for a reality the east already names. These
are not doublets but the fork's own things, entering as themselves:

- **txorbā** — the bazar soup, the morning bowl (Tk. *çorba*; beside *sup*, which stays the
  generic word)
- **txarxô** — the covered market quarter, the lanes of stalls (Tk. *çarşı* < Pers. *čārsū*;
  the *bazar* is an event and a trade, the *txarxô* is a place)
- **dolmô** — stuffed vine-leaves and their kin (Tk. *dolma*)
- **mangāl** — the charcoal brazier (Tk. *mangal*)
- **tamadā** — the elder of the table, the one who rules a feast's toasts (pan-Caucasian /
  Turkic *tamada* — and the proposed title of the seat's head, the register-climbing loan)

## §4 · The fringes

- **Persian/Khorasani (Herat end):** *nān* above; the tea-house word **txaihanā** (Pers.
  *čāy-xāna*, taken through the Turkic conduit like the tea itself). Kept thin — the Herat
  road colors the eastern casual register, it does not run a second quarry.
- **Genoese-Italian (Crimean end):** the old trade residue, maritime and mercantile only —
  **skalā** "stone landing, quay" (Gen. *scala*; beside canon *kājô*), **fondakô** "the
  walled trading inn, the warehouse-court" (Gen. *fondaco*). Two words and the door is
  closed; the Romance learned crust the fork already shares with every standard does the
  rest.

## §5 · The seam — Turkic root, Nelôxi machine

The hybrid morphology test, passed here as in every fork: **bazarôr** "market trader"
(*bazar* + agent *-ôr*, refusing Turkish's own *-cı*), **txaihanôr** "tea-house keeper,"
**dükānôr** "stall-holder"; *bazarôle* "to the bazar," *txarxôl* "in the market lanes" —
Turkic roots declining through the Estonian-leveled cases and packing into Circassian-led
clauses, without the grammar once asking where the root was quarried.

*Register rule of thumb: if the sentence would take* melk *or* moin *on the mainland, it
takes the left column here; if it would take the ledger's word, it takes the right. The
dialogue that runs this table aloud is `dialogue-bazar.md`.*
