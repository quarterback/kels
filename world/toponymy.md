# Nelôxia — Toponymy: How Places Get Named
*Worldbuilding reference · charter §73*

> A place-name encodes **who named the place and why**. Different cultures on Nelôxia's map used
> different naming elements, so a town's name reveals which culture was dominant where. This is
> the toponymic equivalent of the four-quarry system: when naming a new place, reach for the
> elements of the culture that would plausibly have named it, rather than defaulting to Finnic.

## The gravity rule (from §64), restated for names

- **The water and the deep Finnic heartland** → native Finnic elements.
- **The Hanseatic ports** → Low German elements.
- **The coast and islands (Norse charting)** → Scandinavian elements.
- **The Slavic frontier interior** → Slavic elements.
- **The Baltic substrate under the mainland** → Baltic (Latvian/Lithuanian) elements, often
  inherited and half-kept by later namers.

Do NOT Finnicise an interior Slavic town or a Hanseatic port. Compose the name from the right
culture's parts.

## Romance / Latin elements (the church, the charter, the school)

The learned layer — monastery foundations, chartered institutions, scholarly towns. Sparse on
the map (a handful of places, not a region), which is exactly right: these are point-foundations,
not a settlement wave.

| Element | Meaning | Example |
|---|---|---|
| **sant- / sent-** | saint's foundation | (Santamīk "Saint-Friend," a monastery town) |
| **-vīla** | villa, chartered estate | (Novavīla "new-charter-estate") |
| **mônt-** | mount, hill-foundation | (Môntlivrô "book-mount," a scholar-town) |
| **-kolēgi** | college, learned foundation | (the Kēļs Kolēgi itself) |
| **pôrta-** | gate, port-of-entry (formal) | (Pôrtamār "sea-gate," ceremonial name) |

## How to compose a name (procedure for the founder or a naming agent)

1. **Decide who named it.** Where on the map is it? That fixes the culture (gravity rule above).
2. **Pick the feature.** Is it a fortress, a harbor, a river-mouth, a bay, a market? That fixes
   the element (e.g. fortress → *-bôrk* if German, *-grôd* if Slavic, *-pils* if Baltic).
3. **Pick the modifier.** A descriptor (color, size, a founder's name, a resource) goes in front,
   head-final like all Nelôxi compounds (module 04): *Salz-markõt*, *Nôrd-hafõn*, *Petr-ovô*.
4. **Apply the x=[ks] rule (§47) for official/ceremonial forms** if the town warrants a formal
   register name (as Kunislinnô has the charter-name Kunixa).
5. **Keep foreign scars** (§63): a German element keeps German spelling, a Slavic element Slavic.
   Fix only characters that exist in no language (î, û, â → ô / macrons).

The result: a map you can *read as history* — the naming element alone tells you whether Germans,
Slavs, Norse, Balts, or the Finnic natives were the founders of each place.
