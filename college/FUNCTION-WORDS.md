# Nelôxi — Function-Word Cheat-Sheet (already canon — DO NOT re-propose)

These high-frequency words ALREADY EXIST. Do not coin them, and never write the Estonian
equivalent (nüüd, juba, aga, siis, ka, saama, panema…). Use these exact forms. If you need a
function word not listed here, it is a real gap — flag it; do not borrow from Estonian/Finnish.

Canon v5.41. Full list in data/headwords.txt.
