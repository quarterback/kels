# Nelôxi — Translation Worklist (the delegation roadmap)

*The engine of the language is **translating whole texts**: each one forces the language through a
use it hasn't faced and surfaces the missing words faster than anyone can invent them in the abstract.
This is the claimable worklist. Hand an agent `bundles/BUNDLE-translation-agent.md` (the how) plus one
row from here (the what), and it can translate, coin by doctrine, and merge — in parallel with other
agents, without collision.*

## How to use this (for a translating agent)

1. **Claim a row** — put your name/date in Status and commit that change *before* starting, so two
   agents don't take the same text. Prefer **Tier 1** (highest yield, best cultural fit).
2. **Translate the whole text**, not a fragment — the long arc is what surfaces the seam. For a long
   work, claim it in **movements** (e.g. "Fisher tale, Movement II") and mark each done.
3. Follow `BUNDLE-translation-agent.md`: corpus-not-memory, texture doctrine (images over `-ô`),
   non-possessive kinship, register idioms, **run the merge pipeline**, flag ⟨founder-image⟩ concepts.
4. **Mark the row DONE** with the ruling § and the reader file, and add the gap-list seam you leave.

Prefer **public-domain / freely-licensed** source texts (pre-1929, scripture, UDHR, folktale
collections, one's own composition). Note the source. Don't paste long copyrighted texts wholesale.

## What each register is *for* (so you can add rows sensibly)

- **Folktale / fable** — everyday narrative, dialogue, emotion, kinship, nature, moral. The workhorse;
  highest yield of concrete + lived vocabulary. **Native metaphor territory.**
- **Wisdom / scripture** — the deep abstract register (time, fate, vanity, justice, the soul). Where
  the texture doctrine is tested hardest; resist Latin relex, build from the fields.
- **Legal / civic / news** — the genuinely *imported* institutional layer; borrow by quarry (§79).
- **Poetry / song** — meter, imagery, refrain-formulas; grows the poetic register and idiom stock.
- **Procedure / letter** — imperatives, sequence, formulae, the intimate everyday.

Add a row whenever you see a register the corpus hasn't stress-tested. The gap-list from every
finished text names the next words to hunt — follow the seam.
